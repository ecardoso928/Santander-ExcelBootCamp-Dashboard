# Santander-ExcelBootCamp-Dashboard
Dashboard criado para exibir informações de jogos do Xbox , suas assinaturas divididos em EA Play e Minecraft. Feito em excel com aparência de uma aplicação de sistema. Ao alterarmos as opções de filtro de período os valores do gráfico são alterados, bem como os Total Cards. A visualização responde as perguntas de negócios sobre as assinaturas.
